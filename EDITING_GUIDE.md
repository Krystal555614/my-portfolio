# 如何修改網站內容 (Editing Guide)

這份指南將教你如何修改網站上的文字、圖片，並將更新後的內容上傳到 GitHub。

## 1. 修改文字與圖片

所有的網頁內容都在 **`index.html`** 這個檔案裡。你可以用記事本 (Notepad) 或 VS Code 打開它。

### 常見修改位置：

-   **修改名字/標題**：
    搜尋 `<h1 class="hidden">你好，我是 <br>創意開發者</h1>`，將「創意開發者」改成你的名字。

-   **修改關於我**：
    搜尋 `id="about"`，在下方的 `<p>` 標籤內修改自我介紹的文字。

-   **修改技能**：
    搜尋 `id="skills"`，你會看到很多 `<div class="skill-card hidden">`，修改裡面的 `<h3>` (技能名稱) 和 `<p>` (說明)。

-   **修改作品集**：
    搜尋 `id="projects"`，修改 `<article class="project-card hidden">` 內的標題與描述。

## 2. 更新網站 (上傳修改)

修改並存檔後，你的電腦上的網站已經變了，但 GitHub 上的還沒變。你需要執行以下步驟來更新：

1.  開啟 **Git Bash**。
2.  輸入 `cd ` 並將資料夾拖進去，按 Enter (進入資料夾)。
3.  依序執行以下三個指令：

```bash
git add .
git commit -m "更新網站內容"
git push
```

執行完畢後，等待約 1-2 分鐘，重新整理你的網站網址，就會看到更新後的內容囉！
